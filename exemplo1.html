HTML Attributes

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attribute are always specified in the start tag
  • Attributes usually come in name/value pairs like : name = "value"

Example

                    
                         <a href = "https://www.w3schools.com" target="_blank" </a>

The src attribute

The <img> tag is used to embed an imagem in a HTML page. The src attribute specifies the path to the image to be displayed. Should also contain the width and height attributes, which specifies the width and height of the image (in pixels), the alt tags specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to slow connection, or an error in the src attribute, or if the user uses a screen reader:

Example

                
                     <img src="img_girl.jpg" width = "500" height = "600" alt =  "Girl with a jacket">

The style Attribute

The style attribute is used to add styeles to an elment, such as color, font, size, and more.

Example

                    
                         <p style="color:red;font-size: 30px;">This is a red paragraph. </p>

This is a red paragraph with a font-size of 30 pixels

The lang Attributes

You should always include the lang attribute inside the <html> tag, to declare the language of the Web page. This is meant to assist search engines and browsers. The following example specifies English as the language: 
                
                    <!DOCTYPE html>
                    <html lang="en">
                    <body>
                    ...
                    </body>
                    </html>
Country codes can also be added to the language code in the lang attribute. So, the first two characters define the language of the HTML page, and the last two characters define de country. The following example specifies English as the language and Unite States as the country: 
                
                    <!DOCTYPE html>
                    <html lang="en-US">
                    <body>
                    ...
                    </body>
                    </html>

Suggestions

The HTML standard does not requires lowercase attribute names. However, W3C recommends lowercase attributes in HTML, and demandas lowercase attirubes for stricter document types like XHTML. The HTML standard does not required quotes around attribute values. However, W3C recommends quotes in HTML, and demands quotes for stricter document types like XHTML.

Good:

                    
                        <a href="https://www.w3schools.com/html/">Visit our HTML tutorial>

Bad

                    
                        <a href=https://www.w3schools.com/html/>Visit our HTML tutorial>

Single or Double Quotes

Double quotes around attribute vales are the most common in HTML, but singl quotes can also be used.
In some situations, when the attribute value itself contains double quotes, it is necessary to use single quotes, or vice versa: 
                
                    <p title='John "ShotGun" Nelson'></p>
                    <p title="John 'ShotGun' Nelson"></p>
